# Momentum Forge

A polished Expo Router productivity app for iOS, Android, and web. Momentum Forge is offline-first and stores tasks, habits, and profile data locally on the device with AsyncStorage.

## Features

- Create, edit, complete, and delete tasks
- Create habits, group them, and check them off daily
- Open each habit in its own tracker page
- Add notes, due dates, tags, projects, repeat rules, and reminders
- Search tasks and filter by all, active, or completed
- Animated focus board on the home screen
- Activity tracker with a GitHub-style contribution table, group stars, and weekly graph
- Habit and todo activity combined into one daily board
- Local profile setup
- Persistent dark mode setting
- Local task stats, habit stats, completion rate, overdue count, archive, and undo actions
- Works in Expo Go with no backend required

## Get Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Run checks:

```bash
npx tsc --noEmit
npm run lint
```
