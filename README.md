# Focus List

A polished Expo Router task app for iOS, Android, and web. Focus List is offline-first and stores tasks locally on the device with AsyncStorage.

## Features

- Create, edit, complete, and delete tasks
- Create habits and check them off daily
- Add notes, due dates, tags, projects, repeat rules, and reminders
- Search tasks and filter by all, active, or completed
- Animated focus board on the home screen
- Activity tracker with a GitHub-style contribution table and weekly graph
- Habit and todo activity combined into one daily board
- Persistent dark mode setting
- Local task stats, habit stats, completion rate, overdue count, archive, and undo actions
- Works in Expo Go with no backend required

## Get Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Run checks:

```bash
npx tsc --noEmit
npm run lint
```

## Tech Stack

- Expo 54
- React Native 0.81
- Expo Router 6
